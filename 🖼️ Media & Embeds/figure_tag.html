<figure> - Self-Contained Content

🧠 What it does:

  • It represents self-contained content, such as an image, diagram, code listing, or video.
  • The content is related to the main document but can be moved to another part of the page without affecting the flow of the document.
  • The `<figure>` element is a semantic tag that helps group a piece of media with its associated caption.

🕰️ When to use:

  • When creating standalone content that is referenced in the main flow of a document.
  • To semantically group content like images, diagrams, or code with an optional caption to provide context.
  • For improving accessibility and SEO by giving a clear, structured meaning to your media content.

💻 Syntax:

<figure>
  <img src="image.jpg" alt="Description of the image">
  <!-- Optional `figcaption` tag goes here -->
</figure>
