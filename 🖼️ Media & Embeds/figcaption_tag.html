<figcaption> - Figure Caption

🧠 What it does:

  • It represents a caption or legend for the content of its parent `<figure>` element.
  • The tag provides an accessible description for the content within a `<figure>` element.
  • It must be the first or last child of a `<figure>` element.

🕰️ When to use:

  • To add a descriptive text to an image, diagram, code block, or other content wrapped in a `<figure>` tag.
  • To provide contextual information for viewers and screen readers, which helps with accessibility.
  • When a figure's content needs a title or brief explanation to be fully understood.

💻 Syntax:

<figure>
  <img src="image.jpg" alt="Description of the image">
  <figcaption>This is a caption for the image.</figcaption>
</figure>
