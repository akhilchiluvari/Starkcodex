<svg> - Scalable Vector Graphics

🧠 What it does:

  • It defines a container for Scalable Vector Graphics (SVG).
  • Allows for dynamic, scriptable rendering of 2D shapes and bitmap images directly on a webpage.
  • SVG graphics are defined in XML format and do not lose quality when they are resized or zoomed.

🕰️ When to use:

  • To create graphics, animations, and interactive content like logos, charts, and diagrams.
  • When you need images that scale without losing quality for responsive web design.
  • For graphics that are easily editable with a text editor and can be animated with CSS or JavaScript.

💻 Syntax:

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>
