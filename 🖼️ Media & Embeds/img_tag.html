<img> - Image Element

🧠 What it does:

  • Embeds an image into a web page.
  • It is a self-closing tag, meaning it does not have a separate closing tag.
  • Requires the `src` attribute to specify the image source and the `alt` attribute for alternative text.

🕰️ When to use:

  • To display images from various sources, such as files on a website or external URLs.
  • To add visual content and enrich the user experience.
  • To provide alternative text for accessibility and search engine optimization (SEO).

💻 Syntax:

<img src="path/to/your/image.jpg" alt="Description of the image">
