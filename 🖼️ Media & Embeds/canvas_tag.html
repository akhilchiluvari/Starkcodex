<canvas> - Graphics Canvas

🧠 What it does:

  • Provides a resolution-dependent drawing surface for 2D graphics and animations.
  • It acts as a container for graphics, which are then drawn using a scripting language like JavaScript.
  • It is an HTML5 element that can be used to render graphs, game graphics, art, or other visual images on the fly.

🕰️ When to use:

  • When you need to create dynamic and interactive content like games, charts, or photo compositions.
  • To draw 2D shapes, text, or gradients directly on a web page using JavaScript.
  • For rendering graphics on the fly without using an image file.

💻 Syntax:

<canvas id="myCanvas" width="200" height="100">
    Your browser does not support the HTML canvas tag.
</canvas>

🔍 Output:

This is a simple JavaScript example that draws a rectangle on the canvas.
