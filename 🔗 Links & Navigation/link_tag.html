<link> - External Resource Link

🧠 What it does:

  • Defines a relationship between the current document and an external resource.
  • Most commonly used to link external CSS files for styling.
  • Also used for favicons, preloading, or alternate translations of a site.

🕰️ When to use:

  • To include an external stylesheet in your webpage.
  • When setting up favicons for browser tabs.
  • For advanced use cases like preloading assets or linking alternate languages.

💻 Syntax:

<head>
    <link rel="stylesheet" href="styles.css">
</head>

🔍 Output:

No direct visible output, but the page will apply styles from the external CSS file.
