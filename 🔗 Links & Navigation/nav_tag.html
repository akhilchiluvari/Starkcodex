<nav> - Navigation Section

🧠 What it does:

  • Represents a section of navigation links within a webpage.
  • Provides semantic meaning for search engines and assistive tech.
  • Typically used for site menus, tables of contents, or in-page navigation.

🕰️ When to use:

  • For your site’s main navigation menu.
  • When you need secondary nav like footer links or in-article navigation.
  • Use multiple <nav> blocks if needed, but only for actual navigation.

💻 Syntax:

<nav>
    <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="about.html">About</a></li>
        <li><a href="services.html">Services</a></li>
        <li><a href="contact.html">Contact</a></li>
    </ul>
</nav>

🔍 Output:

  • Home
  • About
  • Services
  • Contact
