<base> - Base URL for Links

🧠 What it does:

  • Sets a base URL for all relative URLs in a document.
  • Allows you to define a default href or target for links.
  • Only one <base> element can be used in a document.

🕰️ When to use:

  • When you want all relative links to use the same base path.
  • If you want all links to open in a specific target (like _blank).
  • Useful when hosting content that may move between directories or domains.

💻 Syntax:

<head>
    <base href="https://example.com/" target="_blank">
    <!-- Now all relative links use this base URL -->
</head>

🔍 Output:

No direct visible output, but links like <a href="about">About Us</a> will point to https://example.com/about.
