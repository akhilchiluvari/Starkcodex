<map> & <area> - Image Maps

🧠 What it does:

  • The `<map>` tag defines an image map with clickable areas.
  • The `<area>` tag defines a specific clickable region within the image map.
  • The `<img>` element uses the `usemap` attribute to link to the `<map>` element by its `name` attribute.

🕰️ When to use:

  • To create a single image with multiple interactive regions that link to different destinations.
  • When you need to provide a graphical interface for navigation, such as on a geographical map or an infographic.
  • To provide clickable areas on an image that do not require a separate button for each link.

💻 Syntax:

<img src="planets.jpg" alt="Planets" usemap="#planetmap">
<map name="planetmap">
  <area shape="rect" coords="0,0,82,126" href="sun.html" alt="Sun">
  <area shape="circle" coords="90,58,3" href="mercur.html" alt="Mercury">
  <area shape="poly" coords="124,58,8,6,3" href="venus.html" alt="Venus">
</map>

🔍 Output:

This example requires a local image and links to be fully functional.

The `usemap` attribute of the image links it to the `name` of the map.
