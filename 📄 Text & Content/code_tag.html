<code> - Inline Code Element

🧠 What it does:

  • Represents a piece of computer code (inline, by default).
  • Semantic — informs browsers/screen readers that the content is code.
  • Used inside other tags like <p>, <pre>, <li>, etc.

🕰️ When to use:

  • To display inline code snippets within paragraphs or lists.
  • For referencing file names, functions, commands, or variables.
  • Often paired with <pre> for multi-line code blocks.

💻 Syntax:

<p>Use <code>git status</code> to check your repo state.</p>

🔍 Output:

Use git status to check your repo state.
