<div> - Generic Container

🧠 What it does:

  • Defines a generic block-level container for grouping elements.
  • Has no semantic meaning by itself — purely structural.
  • Often used with CSS or JavaScript to style or manipulate content.

🕰️ When to use:

  • To group related HTML elements for styling or scripting.
  • When no other semantic container tag fits the purpose.
  • For layout building in combination with CSS Grid or Flexbox.

💻 Syntax:

<div class="container">
    <h2>Welcome</h2>
    <p>This is inside a div container.</p>
</div>

🔍 Output:

Welcome

This is inside a div container.
