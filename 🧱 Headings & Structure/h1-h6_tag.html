<h1> - <h6> - Headings

🧠 What it does:

  • Defines headings from most important (<h1>) to least (<h6>).
  • Helps structure the document into a readable hierarchy.
  • Search engines and screen readers use them to understand content flow.

🕰️ When to use:

  • Use <h1> once per page for the main title.
  • Use <h2>-<h6> to structure sections and subsections.
  • To improve accessibility and SEO (search engine optimization).

💻 Syntax:

<h1>This is H1</h1>
<h2>This is H2</h2>
<h3>This is H3</h3>
<h4>This is H4</h4>
<h5>This is H5</h5>
<h6>This is H6</h6>

🔍 Output:

This is H1

This is H2

This is H3

This is H4

This is H5
This is H6
