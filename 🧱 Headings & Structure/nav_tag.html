<nav> - Navigation Section

🧠 What it does:

  • Defines a set of navigation links for the website.
  • Helps users navigate between different sections or pages.
  • Improves semantic meaning for search engines and assistive technologies.

🕰️ When to use:

  • To group primary navigation links in one section.
  • For menus, tables of contents, or quick links.
  • To make website navigation clear and consistent.

💻 Syntax:

<nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
    <a href="#services">Services</a>
    <a href="#contact">Contact</a>
</nav>

🔍 Output:

Home About Services Contact
