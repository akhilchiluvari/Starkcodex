<span> - Inline Container

🧠 What it does:

  • Defines a generic inline container for text or other inline elements.
  • Has no semantic meaning by itself — used mainly for styling.
  • Can apply CSS styles or JavaScript functionality to specific text portions.

🕰️ When to use:

  • To style part of a text without breaking the flow into a new block.
  • When you need to apply CSS to a specific word, phrase, or inline element.
  • For inline JavaScript targeting.

💻 Syntax:

<p>This is a <span style="color: red;">highlighted</span> word.</p>

🔍 Output:

This is a highlighted word.
